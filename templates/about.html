SWViewer is an app to monitor the recent changes of a wiki in real-time. See documentation. Any questions, please ask at SWViewer talk page on metawiki. Be careful, big internet traffic!

Data collection
Appication collects and stores the following data:
  • Username1
  • The Talk messages2 3
  • Date of last access1 3
  • User settings of tool and name of their presets1
  • Stats of user actions via tool (i.e. reverts, undo, edit, warn, RFD, etc)2
  • Logs of on-wiki actions which are performed via tool (i.e. diff, date)2
1 This data may be available to the Wikimedia Foundation, its volunteers and Wikimedia Cloud Services users.
2 This data available to the public.
3 This data deletes in two weeks after last access of tool.
